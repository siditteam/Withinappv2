You are a world-class product designer and experience architect.

Your task is to create the "Learn" section of a meditation app.

IMPORTANT:
- Do NOT redesign the overall app style
- Follow existing typography, colors, spacing system
- Keep everything minimal, premium, and calm
- Focus only on structure, hierarchy, and clarity

--------------------------------------------------

SECTION PURPOSE

The "Learn" section is a structured space where users can explore full teaching series.

It is NOT a content feed.
It should feel like a curated library of wisdom.

Users should feel:
"I am entering a space of depth and learning"

--------------------------------------------------

CONTENT INPUT

There are 9 series.

Each series contains multiple episodes.

Each series represents a complete learning journey.

--------------------------------------------------

STRUCTURE

Create a clean vertical layout with:

1. Header
- Title: Learn
- Optional subtitle: “Structured paths to understanding”

2. Series List (Main Section)

Display all 9 series as cards in a vertical scroll.

Each series card should include:
- Series Title
- Short 1-line description
- Subtle indicator of depth (e.g. Beginner / Intermediate / Deep)
- Number of episodes (optional)
- Visual hierarchy (clear but minimal)

--------------------------------------------------

CARD BEHAVIOR

When user taps a series:
→ open series detail page

--------------------------------------------------

SERIES DETAIL PAGE

Each series opens into:

1. Title + short description

2. Episode List (clean vertical flow)

Each episode item:
- Title
- Optional short line (if available)
- No clutter

3. Progress indicator (subtle)
- e.g. “3 / 12 completed”

--------------------------------------------------

EXPERIENCE FEEL

- Clean
- Quiet
- Intentional
- Premium
- No distractions

Avoid:
- overcrowding
- unnecessary visuals
- heavy UI elements

--------------------------------------------------

ORGANIZATION RULES

- Keep all 9 series equally important
- No algorithmic sorting
- No recommendations here
- This is a pure learning library

--------------------------------------------------

FINAL OUTPUT

Return:
- Learn section layout structure
- Series card structure
- Series detail page structure
- Interaction flow (tap → open → read)

--------------------------------------------------

GOAL

The user should feel:

"This is a place I can return to again and again to learn, slowly and deeply."
